No Admin Account?

Create admin account:

1. Go to Supabase Dashboard → Authentication → Users

2. Click "Add User" with email: admin@kesnnur.org

3. Run SQL: UPDATE profiles SET role = 'super_admin' WHERE email = 'admin@kesnnur.org';
